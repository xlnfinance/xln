# xln 0.1.5

## Release Notes

The first measured xln baseline. It captures the production demo network,
recovery/watchtower stack, bilateral account consensus, durable storage, relay,
P2P transport, and the original release gates.

### Baseline

- Runtime and account consensus hardened for frame replay, ACK recovery, HTLC,
  payment, swap, and dispute flows.
- Watchtower recovery gained encrypted remedies, production endpoints, restart
  resilience, and browser recovery coverage.
- Storage moved to durable-write-before-send ordering with canonical replay
  checks and fail-closed recovery barriers.
- The release was still pre-mainnet and required separate production key,
  chain, gas, monitoring, and launch approval work.
